# 项目介绍

-   采用 `Koishi+Onebot+NapCat`开发的机器人
-   未作多语言适配

## 功能

-   输出测试
-   接入大模型
-   宿舍剩余电量查询
-   获取 OI 比赛信息
-   Clash Royale 相关功能
-   YuGiOh 相关功能
-   输出 VV 梗图
-   输出机器人信息
